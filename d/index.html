Aplikační software

Aplikační software zahrnuje široké spektrum programů, od kancelářských a grafických aplikací přes multimediální a komunikační nástroje až po databázové systémy, antivirové programy a integrovaná vývojová prostředí, které usnadňují specifické uživatelské činnosti.

Definice Základní aplikace Speciální aplikace IDE

Definice a význam

Aplikační software zahrnuje uživatelské programy určené pro specifické účely. Pro jejich spuštění je nezbytný operační systém.

Základní skupiny aplikací

Kancelářské balíky

Kancelářské balíky obsahují nástroje, jako jsou textové procesory, tabulkové procesory, prezentační nástroje a databázové systémy.

  • Komerční: Microsoft Office, Apple iWork
  • Open-source: OpenOffice, WPS Office

Grafické aplikace

Grafické programy umožňují tvorbu a úpravu grafických dokumentů, skenování a práci s digitálními obrazy. Příklady:

  • Profesionální: Adobe Photoshop, Adobe Illustrator
  • Open-source: GIMP, Inkscape

Další oblasti: CAD systémy, DTP programy, software pro animace.

Multimediální programy

Programy pro zpracování a prezentaci audiovizuálních dat:

  • Multimediální přehrávače
  • Editory zvuku a videa
  • Počítačové hry

Specializované aplikace

Programy pro komunikaci v počítačové síti

Tyto aplikace zahrnují webové prohlížeče, e-mailové klienty, software pro online komunikaci (IM) a nástroje pro správu sítí.

Informační a databázové systémy

Informační systémy (IS) poskytují komplexní řešení pro firmy a instituce. Základní komponenty:

  • Datové sklady (Data Warehouse)
  • ERP systémy (Enterprise Resource Planning)

Hlavní firmy: Oracle, Microsoft, IBM, SAP.

Malware a antivirové programy

Malware ohrožuje počítačové systémy, zatímco antivirové programy jako AVG, AVAST, nebo ESET je chrání.

Integrovaná vývojová prostředí (IDE)

IDE slouží programátorům pro efektivní vývoj aplikací. Obsahují:

  • Editor kódu
  • Ladicí nástroje
  • Překladače
