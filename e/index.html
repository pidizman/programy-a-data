Softwarové licence

Software lze získat různými licenčními modely, od komerčního a OEM přes shareware a freeware až po svobodný a open source software, přičemž každý z nich určuje pravidla použití, distribuce a úprav programů.

Uživatel nemůže koupit program jako celek se všemi právy k němu, ale může získat pouze licenci (oprávnění) k jeho užívání. Licence je většinou nepřenosná, a proto není možné tuto licenci někomu dalšímu prodat nebo darovat.

Multilicence

Při koupi programu běžně získává uživatel oprávnění provozovat program na jednom počítači. V případech hromadného využití programu stačí koupit jedno balení programu a vícenásobnou licenci pro další počítače, která bývá cenově výhodnější.

Komerční software

Komerční software je často označován jako proprietární nebo software s uzavřeným kódem. Jeho autor omezuje možnosti jeho používání licencí (typicky EULA). Zdrojové kódy nejsou volně dostupné a nejsou povoleny úpravy či redistribuce díla.

OEM

OEM (Original Equipment Manufacturer) označuje produkty, které jsou určeny k zabudování do jiných zařízení. Například software nabízený společně s hardware za sníženou cenu.

Demoverze a zkušební verze

  • Demoverze: Programy s omezenými funkcemi, které umožňují vyzkoušet funkčnost před koupí.
  • Zkušební verze: Plně funkční programy, ale časově omezené.

Shareware

Program, který lze používat po určitou dobu, po jejímž uplynutí je nutné zaplatit poplatek autorovi.

Freeware

Programy, které jsou zdarma k používání a šíření, avšak s podmínkou dodržování autorských práv.

Public Domain

Programy k volnému užití, jejichž autoři se vzdali určitých práv a umožňují jejich volné šíření a úpravy.

Svobodný software a GNU/Linux

Svobodný software je šířen pod licencí Open Source, například GPL. Myšlenku svobodného softwaru formuloval Richard M. Stallman a projekt GNU/Linux je dnes populární alternativou k drahým komerčním programům.

Open Source a licence GPL

Open Source programy jsou vždy šířeny spolu se zdrojovými texty. Licence GPL stanovuje podmínky, za jakých lze takový software upravovat a distribuovat.
